Проект Movies-explorer (фронтенд)

Проект является дипломной работой на курсе веб-разработчик Яндекс.Практикума.

Используемые технологии:
React
JS
HTML
CSS 


Запуск проекта:
npm i — установка зависимостей

npm run start — запускает приложение
